## Installing Coplay

Follow these simple steps to install Coplay in your Unity Editor and start enhancing your game development workflow.

### Installation Steps

<AccordionGroup>
  <Accordion title="Prerequisites">
    Before installing Coplay, ensure you have:

    - [Unity Editor](https://unity.com/download) installed on your machine
    - Git installed on your system (required for package installation)
      - Installation Instructions: https://github.com/git-guides/install-git
    - An active internet connection

  </Accordion>
  <Accordion title="Install via Package Manager" icon="box-open">
    1. Open your Unity project
    2. Go to **Window \> Package Manager** in the Unity Editor menu
    3. Click the **\+** icon in the top-left corner of the Package Manager window
    4. Select **Add package from git URL...**
    5. Paste the appropriate installation URL according to your Unity version:

       <Info>
         For Unity 2022 and above:
         https://xxx@github.com/CoplayDev/unity-plugin.git#beta
       </Info>
       <Info>
         For Unity 2021 (a few UI warnings will be logged that can be safely ignored);  https://xxx@github.com/CoplayDev/unity-plugin.git#beta-unity-2021
       </Info>
       <Warning>
         Note: Unity 2021 and lower is not part of our long-term support plan and there might be some unknown bugs in Unity 2021.
       </Warning>
    6. Click **Add** and wait for the installation to complete (this may take a
       minute)

  </Accordion>
    <Accordion title="Verify Installation">
    After installation, you should see the Coplay icon in your Unity Editor's toolbar. If you don't see the Coplay menu or encounter any issues during installation, please visit our [Community Discord](http://discord.com/invite/KmGfEr6d3v) for assistance.
  </Accordion>
  <Accordion title="First-time Setup">
    1. You'll see the **Coplay** menu appear in your Unity Editor's top menu bar.
    2. Click on **Coplay** or use the shortcut **Command\+G** (Mac) / **Control\+G** (Windows) to open the Coplay window.
    3. Click the **Sign In** button in the Coplay panel.
    4. A web browser will open for you to complete the authentication process.
    5. Once authenticated, you'll returnto Unity with Coplay ready to use.
  </Accordion>
</AccordionGroup>
