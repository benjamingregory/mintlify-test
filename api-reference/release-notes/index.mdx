## May 2023

### May 15, 2023 - Version 0.4.0

- **New Feature**: Added support for multiple AI models
- **Improvement**: Enhanced scene analysis capabilities
- **Bugfix**: Fixed authentication issues on macOS
- **Improvement**: Reduced memory usage during large project analysis

### May 3, 2023 - Version 0.3.2

- **Bugfix**: Resolved UI scaling issues on high-DPI displays
- **Improvement**: Better syntax highlighting in code suggestions
- **New Feature**: Added shortcuts for common Unity-specific queries

## April 2023

### April 20, 2023 - Version 0.3.0

- **New Feature**: Added context-aware code completion
- **Improvement**: Faster response times for complex queries
- **Bugfix**: Fixed integration with Unity 2022.1+
- **New Feature**: Expanded documentation references

### April 5, 2023 - Version 0.2.5

- **Bugfix**: Resolved startup crashes on Windows
- **Improvement**: Better handling of large Unity projects
- **New Feature**: Added support for code refactoring suggestions

## March 2023

### March 25, 2023 - Version 0.2.0

- **New Feature**: Beta release of asset optimization suggestions
- **Improvement**: Enhanced UI with collapsible panels
- **Bugfix**: Fixed authentication token refresh issues
- **New Feature**: Added dialog history persistence

### March 10, 2023 - Version 0.1.5

- **Improvement**: Better handling of C# syntax in responses
- **Bugfix**: Fixed issues with window resizing
- **New Feature**: Added ability to export conversation history

## February 2023

### February 28, 2023 - Version 0.1.0

- **Initial Beta Release**: First closed beta version

  - Interactive chat interface
