Coplay offers numerous settings to customize its behavior and appearance within your Unity Editor. This guide explores all available configuration options to help you optimize Coplay for your workflow.

## Accessing Settings

You can access Coplay settings in two ways:

1. Click on the gear icon (⚙️) in the Coplay panel header
2. Navigate to **Unity Menu → Coplay → Settings**

## General Settings

<ResponseField name="Model Selection" type="dropdown">
  Choose which AI model powers your Coplay assistant.
  <Expandable title="Available Models">
    <ResponseField name="Standard" type="default">
      Balanced performance and capabilities for everyday Unity development
      tasks.
    </ResponseField>
    <ResponseField name="Performance">
      Faster response times but potentially less nuanced answers. Ideal for
      quick queries and code suggestions.
    </ResponseField>
    <ResponseField name="Advanced">
      More sophisticated understanding and code generation capabilities, but may
      have slightly slower response times.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="Response Length" type="slider">
  Control how detailed Coplay's responses should be. - **Concise**: Brief,
  to-the-point responses - **Balanced**: Moderate level of detail (default) -
  **Detailed**: Comprehensive explanations with examples
</ResponseField>

<ResponseField name="Code Style" type="dropdown">
  Set your preferred code formatting style for generated code.
  <Expandable title="Options">
    <ResponseField name="Unity Standard">
      Follow Unity's recommended coding conventions
    </ResponseField>
    <ResponseField name="Personal">
      Automatically adapt to match your existing code style
    </ResponseField>
    <ResponseField name="Custom">
      Define your own preferences for braces, indentation, etc.
    </ResponseField>
  </Expandable>
</ResponseField>

## Interface Settings

<ResponseField name="Theme" type="dropdown">
  Choose how Coplay appears in your Unity Editor. - **Light**: Bright interface
  with dark text - **Dark**: Dark interface with light text (default) -
  **System**: Match your system preferences - **Unity**: Match your Unity Editor
  theme
</ResponseField>

<ResponseField name="Font Size" type="slider">
  Adjust the text size in the Coplay panel. - Small (10pt) - Medium (12pt) -
  Default - Large (14pt) - Extra Large (16pt)
</ResponseField>

<ResponseField name="Window Position" type="dropdown">
  Choose where Coplay's panel appears in the Unity Editor. - **Floating**:
  Movable window that can be placed anywhere - **Right Dock**: Docked on the
  right side of the Unity Editor (default) - **Left Dock**: Docked on the left
  side of the Unity Editor - **Bottom Dock**: Docked at the bottom of the Unity
  Editor
</ResponseField>

<ResponseField name="Stay Open" type="boolean" default="true">
  When enabled, Coplay's panel will remain open when switching between different
  Unity windows and panels. When disabled, Coplay will automatically minimize
  when you click elsewhere in the Unity Editor.
</ResponseField>

## Behavior Settings

<ResponseField name="Context Awareness" type="dropdown">
  Control how much information from your Unity project Coplay considers when
  providing responses.
  <Expandable title="Options">
    <ResponseField name="Minimal">
      Only consider the current conversation and basic Unity context
    </ResponseField>
    <ResponseField name="Standard" type="default">
      Include the active GameObject, component details, and recent scripts
    </ResponseField>
    <ResponseField name="Maximum">
      Consider broader project context, including asset references,
      dependencies, and project settings
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="Auto-Suggestions" type="boolean" default="true">
  When enabled, Coplay will automatically provide suggestions based on your
  current work context in Unity.
</ResponseField>

<ResponseField name="Keyboard Shortcut" type="key combination">
  Customize the keyboard shortcut to toggle Coplay's panel. Default:
  **Command+G** (Mac) / **Control+G** (Windows)
</ResponseField>

<ResponseField name="Session History" type="dropdown">
  Configure how Coplay handles your conversation history.
  <Expandable title="Options">
    <ResponseField name="Keep Forever">
      Maintain all conversation history between Unity Editor sessions
    </ResponseField>
    <ResponseField name="Clear on Exit" type="default">
      Start fresh conversations each time you restart Unity
    </ResponseField>
    <ResponseField name="Time-Based">
      Clear conversations older than a specified time period
    </ResponseField>
  </Expandable>
</ResponseField>

## Advanced Settings

<ResponseField name="API Key" type="string">
  For Professional and Enterprise users, enter your Coplay API key here to
  enable headless usage and custom integrations.
  <Warning>
    Keep your API key private. It gives access to your Coplay account. Only
    available on Professional and Enterprise plans.
  </Warning>
</ResponseField>

<ResponseField name="Local Processing" type="boolean" default="false">
  When enabled, some of Coplay's features will use local models for enhanced
  privacy and reduced latency.
  <Note>
    This feature requires additional downloads and may reduce some capabilities.
    Available on Professional and Enterprise plans.
  </Note>
</ResponseField>

<ResponseField name="Logging Level" type="dropdown" default="Errors">
  Configure how verbose Coplay's internal logging should be. - **None**: No
  logging - **Errors**: Log only errors - **Standard**: Log errors and important
  operations - **Verbose**: Detailed logging for troubleshooting
</ResponseField>

## Reset Settings

At the bottom of the settings panel, you'll find two options:

1. **Restore Defaults**: Reset all settings to their original values
2. **Export Settings**: Save your current settings configuration to share across projects or team members
